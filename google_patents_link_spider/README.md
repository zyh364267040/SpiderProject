# google_patents_link_spider

Google Patents 专利搜索结果链接采集脚本整理版。

## 文件说明

- `google_patents_link_spider.py`：请求 Google Patents 搜索接口，按关键词和语言条件翻页采集专利详情链接，并保存到文本文件。

## 说明

这是早期专利搜索结果链接采集代码的整理归档版本。整理时只保留核心代码，没有迁移历史结果文件 `google.txt`。

当前代码中的搜索关键词、语言条件和页数仍是旧脚本写死的参数，后续如需复用，建议改成可配置参数，并进一步采集专利标题、摘要、申请人、公开日期等详情字段。
