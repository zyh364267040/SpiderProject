# samr_national_standard_spider

国家标准信息公共服务平台 SAMR 国家标准信息采集脚本整理版。

## 文件说明

- `samr_national_standard_spider.py`：按关键词分页检索国家标准，进入标准详情页提取字段，并写入 Excel。

## 当前采集字段

- 标准号
- 标准名称
- 标准类别
- 批准日期
- 实施日期

## 整理说明

当前代码的搜索关键词为“智能”，搜索范围为国家标准，分页范围为第 1 至 24 页。代码会先提取搜索结果中的标准 ID，再进入详情页采集具体信息。

原代码预留了行业领域、状态、备案号、备案日期等 Excel 表头，但尚未实现这些字段的解析。后续重新运行前，应根据当前网站页面结构检查 XPath，并将关键词和页数改为可配置参数。
