# csres_dl_standard_spider

中国标准服务网 CSRES 电力行业 DL 标准信息采集脚本整理版。

## 文件说明

- `csres_dl_standard_spider_1.py`：早期分页采集版本，包含列表页、详情页和替代情况字段解析逻辑。
- `csres_dl_standard_spider_2.py`：较完整版本，包含 Excel 断点续爬、字段解析和结果保存逻辑。

## 采集字段

- 标准编号
- 标准名称
- 发布部门
- 实施日期
- 状态
- 替代情况

## 说明

这是早期行业标准信息采集代码的整理归档版本。整理时只保留核心代码，历史 Excel 结果文件、HTML 缓存和临时测试脚本不提交到 GitHub。

代码依赖旧网页结构，后续如需重新运行，需要根据当前网站页面结构检查 XPath、分页范围和请求间隔。
