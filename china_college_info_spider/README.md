# china_college_info_spider

中国高校公开信息异步采集案例。项目通过中国教育在线公开数据接口获取高校基础信息、学校层次、国内外排名、地区以及招生办公开联系方式，并保存为 CSV。

## 采集字段

- 学校 ID、学校名称、学校层次
- 软科、校友会、武书连排名
- QS 世界排名、US 世界排名
- 学校类型、省份、城市、地区
- 招生办公开电话、招生办官网

## 功能特点

- `aiohttp` 与 `asyncio` 异步采集
- 共用一个 `ClientSession`
- 默认并发量限制为 20
- 请求失败自动重试
- 根据学校 ID 断点续采
- 缺失字段安全容错
- 所有任务完成后统一写入 CSV，避免并发写文件

## 使用方法

安装依赖：

> pip install -r requirements.txt

运行：

> python china_college_info_spider.py

结果默认保存到脚本所在目录的 `college_info.csv`。该结果文件已被仓库 `.gitignore` 中的通用 CSV 规则排除，不应提交包含公开联系电话的历史采集结果。

## 整理说明

本项目由2022年的历史脚本整理而来。整理时未迁移包含2684所学校信息的历史 CSV 和缓存文件，并修复了硬编码旧路径、每次请求创建Session、500并发、关闭SSL校验、裸异常处理、字段缺失报错和协程并发写CSV等问题。

请遵守数据来源网站的服务条款，控制请求频率，并妥善处理招生办公开电话等联系信息。
