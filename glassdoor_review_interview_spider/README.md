# glassdoor_review_interview_spider

Glassdoor 公司评论与面试经验采集脚本整理版。

## 文件说明

- `glassdoor_review_spider.py`：公司 Reviews 评论采集脚本。
- `glassdoor_interview_spider.py`：公司 Interviews 面试经验采集脚本。

## 说明

这是早期 Glassdoor 采集练习代码的整理归档版本，只保留核心代码文件。原目录中的 URL 列表和历史采集结果没有迁移到 GitHub，避免把批量链接和采集数据混入代码仓库。

代码中仍有一些早期脚本痕迹，例如 Selenium Driver 路径、页面 XPath/CSS 选择器、输出字段格式等，需要按当前运行环境和目标网页结构调整后再使用。
