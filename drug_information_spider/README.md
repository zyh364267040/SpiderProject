# drug_information_spider

多来源药品公开信息采集案例。脚本从 RxList 获取软胶囊类药品条目和详情说明，再结合 DrugFuture 与药智网查询药品规格及生产企业，并将结构化结果保存为 CSV。

## 采集字段

- 药品名称
- 药品说明或处方组成
- 药品规格
- 生产企业

## 技术栈

- `requests`：网页请求和表单查询
- `lxml` / XPath：列表页与详情页解析
- `openpyxl`：Excel 处理实验代码
- CSV：结构化结果保存

## 整理说明

本目录只保留核心代码，没有迁移历史 CSV、Excel、失败链接、缓存文件或 `.DS_Store`。整理时将失败链接记录改为追加写入，避免后一次失败覆盖此前记录。

该脚本为历史学习案例，目标网站结构和访问规则可能已经变化。重新运行前应核对当前页面结构，遵守网站服务条款并控制请求频率。采集结果仅可用于合法的数据研究和编程学习，不能替代专业医疗建议。
