# haodf_doctor_spider

好大夫在线医生公开信息采集脚本整理版。

## 文件说明

- `haodf_doctor_spider.py`：按地区分页采集妇产科医生公开职业信息，并保存为 CSV。

## 采集字段

- 医生姓名
- 性别
- 职称
- 医院名称
- 医院公开联系电话
- 医院地址
- 门诊时间

## 整理说明

整理时只保留核心代码，没有迁移历史 CSV、失败记录和含具体医院排班数据的测试脚本。代码中未保留任何历史登录凭据。

本次整理删除了具体医生页面示例，启用 HTTPS 证书验证，修复了分页 URL 格式化和调试 `break` 导致只采集第一页的问题，并将失败链接记录改为追加写入。

本项目仅用于公开网页信息采集学习。重新运行前应核对网站当前页面结构、访问规则和个人信息保护要求，不应采集或传播非公开医疗与个人信息。
